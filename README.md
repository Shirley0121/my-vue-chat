# AI Экзаменационная система

Веб-приложение на Vue 3 + PHP, интегрированное с DeepSeek API. Генерирует вопросы по C, оценивает ответы, поддерживает переключение языков (русский, китайский, английский).

---

## Структура проекта
my-chat-app/
├── src/ # Исходный код фронтенда
│ ├── locales/ # Файлы языков (zh.js, ru.js, en.js)
│ ├── components/ # Компоненты (LanguageSwitcher.vue)
│ └── App.vue
├── public/
├── api.php # PHP бэкенд (вызов DeepSeek API)
├── .env # Переменные окружения для разработки
├── .env.production # Переменные окружения для продакшна
└── README.md

---

## Локальная разработка (фронтенд + Python бэкенд)

> Для удобства локальной отладки можно использовать Python бэкенд (без установки PHP).

1. Запустите бэкенд (Python)
   ```bash
   cd ../server-deepseek   # предполагается, что папка бэкенда на уровень выше
   python app.py
2.Запустите фронтенд
cd my-chat-app
npm install
npm run dev
3.Откройте браузер: http://localhost:5173
Продакшн-деплой (PHP бэкенд)
1.Установите переменную окружения в файле .env.production:
VITE_API_URL=https://ваш-домен/api.php
2.Соберите фронтенд:
npm run build
3.Загрузите папку dist/ и файл api.php на сервер (корневая директория сайта, например /var/www/html/).
4.Убедитесь, что на сервере установлен PHP и включено расширение cURL.
Переменные окружения
VITE_API_URL – адрес бэкенда (локально: http://localhost:8080/api/chat, на сервере: https://ваш-домен/api.php)
Функционал
✅ Переключение языков (русский, китайский, английский) – сохраняется в localStorage

✅ Автоматическая генерация 10 вопросов по C (один вопрос за раз)

✅ Автоматическая оценка ответов, вывод результата в JSON

✅ История диалога сохраняется в sessionStorage

✅ Markdown-рендеринг и защита от XSS (DOMPurify)

Технологии
Фронтенд: Vue 3, Vite, Markdown-it, DOMPurify

Бэкенд: PHP 8 (cURL) или Python Flask

AI: DeepSeek API
